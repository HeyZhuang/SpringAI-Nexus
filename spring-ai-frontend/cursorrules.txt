以下是为"Spring AI MCP服务HRAG知识库 实战"页面设计的高端美化提示词，延续深空灰(#0F172A) + 鎏金(#D4AF37) + 羊皮纸白(#F8F5F0)的奢华风格，结合"古籍秘阁"概念：

设计概念
prompt
打造"天工开物"式知识检索体验，将古籍藏书阁与现代AI技术融合：
1. 核心隐喻：知识库=皇家秘阁，搜索=星象推演
2. 设计语言：新中式科技风
3. 材质系统：
   - 背景：动态星图(#0F172A) + 鎏金星轨
   - 面板：羊皮纸卷轴 + 青铜镶边
   - 交互：占星罗盘元素
页面结构设计
prompt
1. 头部导航：
   - 左侧：青铜简牍造型标题"Spring AI MCP服务HRAG知识库 实战"
   - 右侧：八卦导航菜单（历史/收藏/设置）

2. 主功能区：
   - 搜索区：星盘造型输入框
   - 结果区：悬浮竹简展示架
   - 操作区：青铜器造型控制面板

3. 状态指示区：
   - 文件上传：青铜鼎造型上传区
   - 系统提示：龟甲占卜式状态卡
视觉规范
搜索区设计
prompt
1. 标题设计：
   - "畅销搜索" → "星象推演"
   - 字体：篆书鎏金字 + 青铜底纹

2. 搜索输入框：
   - 造型：砚台形凹槽
   - 占位符："请输入内容..." → "以天机为墨，书所求之事..."
   - 特效：输入时底部浮现星图轨迹

3. 搜索按钮：
   - 造型：青铜司南指针
   - 状态：未输入时指向随机方位，输入后指向搜索结果方位
结果展示区
prompt
1. 结果容器：
   - 造型：悬浮竹简卷轴
   - 展开特效：点击后如古卷展开，浮现鎏金批注

2. 内容展示：
   - 标题：金文篆刻样式
   - 摘要：竖排仿刻本字体
   - 元数据：青铜简牍标签（日期/来源）

3. 交互特效：
   - 悬停：竹简边缘泛起鎏金光晕
   - 选中：卷轴自动卷起加青铜镇纸
文件上传区
prompt
1. 主容器：
   - 造型：三足青铜鼎
   - 状态：鼎内鎏金液体表示上传进度
   - 提示："投简入鼎，可窥天机"

2. 操作说明：
   - "通过文档 查看主卡" → 玉琮造型按钮
   - 格式提示："玉牍规制：HCA文书，百钱之重(100KB)"
动效系统
prompt

动效实现
javascript
// 搜索司南指向动画
function updateCompassNeedle(angle) {
  const needle = document.querySelector('.needle');
  needle.style.transform = `rotate(${angle}deg)`;
}

// 文件上传动画
function updateUploadProgress(percent) {
  const liquid = document.querySelector('.liquid-fill');
  liquid.style.height = `${percent}%`;
  
  if(percent >= 100) {
    document.querySelector('.cauldron').classList.add('glow');
  }
}

// 竹简展开效果
function toggleScroll(id) {
  const scroll = document.querySelector(`#scroll-${id}`);
  scroll.classList.toggle('expanded');
}
